Bootstrap 4

NOTE: Bootstrap 4 already has it's own spacing library and you should use it instead. You can find it here.

This is a Spacing CSS build specifically for Bootstrap 4

Spacing class changes

In the Bootstrap build, spacing classes are almost the same as the regular build - but the media queries have been changed.

Media queries

The Bootstrap 4 build uses Bootstrap 4 default media queries. This means Spacing CSS will work out of the box with the Bootstrap grid system.

sm - min-width: 576px
md - min-width: 768px
lg - min-width: 992px
xl - min-width: 1200px

This will apply a top padding of 1rem on medium screens and above. On small screens the column will become full width and the padding will be removed.

</div>
