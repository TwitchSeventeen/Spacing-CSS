Gulp

Gulp commands are found in gulpfile.js

default

Starts a file watcher, runs the gulp sass comand

sass

Compiles all .scss files and outputs them into the /dist directory. These files are not compressed and should not be used in production.

minify

Compiles all .scss files, minifys them and outputs them into the /dist directory. These files are ready for production use.
