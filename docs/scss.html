SCSS Variables

Default SCSS variables can be found in the spacing.scss file

$space

The length of a single unit of margin or padding. By default it is 1rem.

$space: 1rem;

$multipliers

An array of multipliers. By default it is (2,3,4,5)

Each multiplier should be a number.

$multipliers: (2,3,4,5);

$sizes

An array of sizes.

Each size consists of an array with the size name and a media query

$sizes: (
  (small, '(min-width: 320px) and (max-width: 480px)'),
  (medium, '(min-width: 481px) and (max-width: 767px)'),
  (large, '(min-width: 768px) and (max-width: 1024px)'),
  (xlarge, '(min-width: 1025px) and (max-width: 1280px)'),
  (xxlarge, '(min-width: 1281px)')
);

$delimiter1

The delimeter used between the property/direction and size

$delimiter1: '-';

$delimiter2

The delimeter used between the size and multiplier

$delimiter2: '-';
